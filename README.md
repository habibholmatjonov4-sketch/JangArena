# Boxing & UFC Bot 🥊

Bu Telegram bot - boks va UFC janglarini ko'rish uchun mo'ljallangan platforma. Foydalanuvchilar bepul va pullik janglarni tomosha qilishlari mumkin.

## 🚀 Xususiyatlar

- 👥 Foydalanuvchi ro'yxatdan o'tish (ism va telefon raqam)
- 🌐 Ko'p tillilik (O'zbek, Rus, Ingliz)
- 👑 Boks va UFC bo'limlari
- 🆓 Bepul janglar
- 💰 Pullik janglar va to'lov tizimi
- 👨‍💼 Admin panel
- 📊 Statistika
- 📢 Xabar yuborish (broadcast)
- 💭 Fikr-mulohazalar

## 📋 Talablar

- Python 3.9+
- Telegram Bot Token
- SQLite3
